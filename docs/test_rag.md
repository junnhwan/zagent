# RAG 联调测试文档

用于快速验证：**文档上传、切块入库、标签检索、答案准确性、幻觉控制**。

## 1. 测试知识文本

将下面内容保存为一个文本文件后上传，知识标签建议填：`客服规则`

```text
账号注册与登录：
- 短信验证码有效期 5 分钟。
- 每日最多输错密码 5 次，超出后锁定 1 小时。
- 忘记密码可通过“手机号 + 验证码”重置，重置后原登录状态全部失效。

订单支付规则：
- 订单提交后需在 15 分钟内完成支付，超时自动取消。
- 支持微信支付、支付宝、银行卡支付。
- 单笔支付金额上限为 5000 元。

物流与配送：
- 工作日 16:00 前下单当日发货，16:00 后次日发货。
- 同城 1 天，省内 2 天，跨省 3~5 天。

售后政策：
- 签收后 7 天内无理由退货，15 天内质量问题可换货。
- 虚拟商品一经售出不予退换。
- 微信/支付宝退款 1~3 个工作日到账，银行卡 3~7 个工作日到账。

会员权益：
- 普通会员 9.8 折，年费会员 9 折且免运费。
- 积分 100 分 = 1 元。
- 积分有效期 1 年。
- 年费会员价格 99 元/年，开通后不可退订。
```

## 2. 核心测试题

### 2.1 命中数字规则

- **提示词**：`根据“客服规则”知识库，订单支付超时多久会自动取消？`
- **重点观察**：是否准确回答 `15 分钟`，不要答成模糊表述。

### 2.2 命中多字段信息

- **提示词**：`根据“客服规则”知识库，密码输错几次会锁定？锁定多久？`
- **重点观察**：是否同时答出 `5 次` 和 `1 小时`。

### 2.3 命中否定规则

- **提示词**：`根据“客服规则”知识库，虚拟商品可以退换吗？`
- **重点观察**：是否明确回答 `不予退换`，不要含糊其辞。

### 2.4 命中复合规则

- **提示词**：`根据“客服规则”知识库，年费会员多少钱？能退吗？`
- **重点观察**：是否同时答出 `99 元/年` 和 `不可退订`。

## 3. 幻觉控制测试

- **提示词**：`根据“客服规则”知识库，系统支持京东支付吗？`
- **重点观察**：应回答知识库未提及，不要自己编造。

## 4. 联调顺序

1. 先上传测试文档，确认返回成功。
2. 先测 **数字规则** 和 **多字段信息**。
3. 再测 **否定规则** 和 **复合规则**。
4. 最后测 **幻觉控制**。
