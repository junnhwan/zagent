# 智能客服知识库测试文档

用于 RAG 知识库导入、向量检索、答案生成与幻觉控制测试。

## 1. 账号注册与登录

- 新用户需使用手机号完成注册，短信验证码有效期为 **5 分钟**。
- 每日最多允许输错密码 **5 次**，超出后账号将锁定 **1 小时**。
- 支持微信、支付宝快捷登录，登录后可绑定手机号完成实名认证。
- 忘记密码可通过“手机号 + 验证码”重置，重置后原登录状态会全部失效。

## 2. 订单支付规则

- 系统支持 **微信支付、支付宝、银行卡支付** 三种方式。
- 订单提交后需在 **15 分钟内** 完成支付，超时将自动取消。
- 单笔订单支付金额上限为 **5000 元**，超过需分单支付。
- 支付成功后不支持立即退款，需在订单发货前申请，审核周期为 **1~3 个工作日**。

## 3. 物流与配送

- 普通商品默认使用 **中通快递**，大件商品使用 **德邦物流**。
- 工作日 **16:00 前** 下单当日发货，**16:00 后** 下单次日发货。
- 同城配送时效为 **1 天**，省内 **2 天**，跨省 **3~5 天**。
- 不支持指定具体送达时间，仅可查看物流实时轨迹。

## 4. 售后政策

- 商品签收后 **7 天内无理由退货**，**15 天内质量问题可换货**。
- 退货需保持包装完好、配件齐全，人为损坏不在保修范围内。
- 退款金额将原路返回，微信 / 支付宝 **1~3 个工作日到账**，银行卡 **3~7 个工作日到账**。
- 虚拟商品（充值卡、会员、兑换码）一经售出 **不予退换**。

## 5. 会员权益

- 普通会员购物享 **9.8 折**，年费会员享 **9 折** 且 **免运费**。
- 会员积分 **100 分 = 1 元**，下单时可直接抵扣现金。
- 积分有效期为 **1 年**，过期自动清零，不可恢复。
- 年费会员价格为 **99 元 / 年**，开通后 **不可退订**。

## 配套测试题

以下问题可直接用于测试 **检索 + 生成**：

1. 订单支付超时多久会自动取消？
2. 密码输错几次会锁定账号？锁定多久？
3. 虚拟商品可以退换吗？
4. 会员积分多少等于 1 元？有效期多久？
5. 退款一般多久到账？
6. 下午 4 点后下单什么时候发货？
7. 年费会员价格是多少？能退吗？
8. 售后多少天内可以无理由退货？

## 建议测试点

导入后可重点观察以下能力是否正常：

- **文本切块** 是否合理
- **向量检索** 是否能命中正确片段
- **答案生成** 是否准确引用原文信息
- **幻觉控制** 是否避免乱编
- **数字、时间、规则类信息** 是否稳定准确
